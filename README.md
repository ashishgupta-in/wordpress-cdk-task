## **Steps for deployment :-**

**`Step-1:`** 
> * Create ***`.env`*** file containing the values for environment variables. <br>
> * Refer the template file ***`env-template.txt`***. 

**`Step-2:`**
> Execute the deployment script file ***`deploy.sh`***. <br>
```ruby
$ bash deploy.sh
```
